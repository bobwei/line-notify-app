## Getting Started

- Config ENV
- Install Node Dependencies
- Start


### Config ENV

create env from example
```
mkdir .envs
cp .example.env .envs/.dev.env
```

link to env
```
ln -s .envs/.dev.env .env
```

### Install Node Dependencies

```
yarn
```

### Start

```
yarn start
```
